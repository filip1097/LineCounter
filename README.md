# LineCounter

LineCounter is able to tell how many blank lines, code lines and comment lines
an input file contains.

### Functionality

The tool currently supports the following languages:

* Antlr     *(.g4)*
* CUP       *(.cup)*
* JavaCC    *(.jjt, .jj)*
* JFlex     *(.flex)*
* MetaLexer *(.mll, .mlc)*
